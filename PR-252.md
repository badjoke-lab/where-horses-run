# PR-252: Country and track public list migration

## Summary

This PR performs roadmap PR-8: move country and racecourse timetable sections to the public timetable list.

## Changed

- Add `ContextTimetableTable.astro` for scoped timetable displays.
- Add country and racecourse filters to `src/data/timetableMeetingRows.ts`.
- Feed `siteData.generated.timetables` from `data/generated/timetable/public/meeting-list.json` for country/racecourse legacy compatibility.
- Render English and Japanese country upcoming-meeting sections from the public timetable adapter.
- Render English and Japanese track upcoming-meeting sections from the public timetable adapter.
- Add `scripts/check-country-track-public-list.mjs`.

## Result

Country and racecourse timetable sections now read from public meeting-list.

English and Japanese country pages use ContextTimetableTable.

English and Japanese track pages use ContextTimetableTable.

The table keeps one row per meeting.

Detail links and official source links come from public view data.

Public rank display rules are shared with the calendar/list pages.

## Scope

No meeting detail page is migrated in this PR.

No legacy input file is deleted.

No source fetching or snapshot acquisition is added.

No racecard, entry, odds, result, payout, prediction, or tip data is displayed.
