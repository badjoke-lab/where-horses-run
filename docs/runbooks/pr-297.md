## Public-safe storage

Only source capability, meeting-date and racecourse confirmation, timing availability, official URLs, review dates, and rank decisions are stored.

The repository does not store or publish participant lists, horses, jockeys, trainers, owners, weights, odds, results, payouts, predictions, full racecards, raw source text, embedded video, or direct stream URLs.
