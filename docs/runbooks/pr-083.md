# PR-083 — Correct UAE racecourse inventory from official evidence

## Summary

PR-083 corrects the UAE / Emirates Racing Authority racecourse inventory using the official Emirates Racing evidence recorded in PR-082. The correction keeps the existing UAE coverage posture unchanged: UAE remains an out-of-window seed / season-gap country for active-window purposes, not broad coverage and not complete coverage.

## Files

- `data/static/country-racing-inventory.json`
- `scripts/check-uae-official-racecourse-inventory.mjs`
- `package.json`
- `docs/runbooks/pr-083.md`

## Official evidence used

PR-083 uses the official source evidence recorded in `docs/runbooks/existing-countries-official-source-evidence.md`:

- Emirates Racing racing-in-the-UAE page: `https://emiratesracing.com/about-us/racing-in-the-uae`
- Emirates Racing 2025-2026 schedule announcement: `https://emiratesracing.com/news/the-emirates-racing-authority-era-has-announced-the-official-schedule-for-the-2025-2026-uae-horse-racing-season-which-is-set-to-begin-on-25th-october-2025-at-the-abu-dhabi-turf-club`
- Emirates Racing current-season calendar source candidate: `https://emiratesracing.com/season-calendar/current-season`

The evidence-backed UAE venue concepts are:

- Meydan
- Jebel Ali
- Abu Dhabi
- Sharjah
- Al Ain

## Data correction made

The UAE / ERA racecourse inventory now includes all five official evidence-backed venue concepts:

- Kept `Meydan`.
- Added `Jebel Ali Racecourse`.
- Kept `Abu Dhabi Turf Club` and added a note that naming needs normalization review against `Abu Dhabi` / `Abu Dhabi Turf Club` / `Abu Dhabi Equestrian Club` wording.
- Added `Sharjah Racecourse / Sharjah Longines Racecourse`.
- Kept `Al Ain`.

The UAE coverage notes now explicitly state that PR-082 official evidence showed five venue concepts and that PR-083 corrected the missing Jebel Ali and Sharjah inventory items.

## What this does not do

PR-083 does not:

- add candidate timetable records;
- add active-window UAE records;
- replace a public overlay;
- add live fetch runtime;
- add a source page parser;

- claim UAE complete coverage;
- claim UAE broad coverage;
- claim UAE active calendar coverage;
- claim UAE 30-day calendar coverage;
- add dependencies.
