# PR-284 — Define the 98-country page completion programme

## Summary

This PR establishes the formal completion contract and canonical tracker for
all 98 country and region detail pages.

It adds:

- a fixed 98-row tracker derived from the seven scope groups in the v0
  specification,
- explicit programme and acquisition states,
- separate date meanings for source checks, evidence review, tested meetings,
  profile review and publication,
- a validator for counts, identities, status transitions, dates and note
  references,
- a dedicated GitHub Actions workflow,
- a correction that separates Zimbabwe's evidence review date from the future
  tested meeting date.

## Baseline after this PR

- `note_reviewed`: 12
- `profile_ready`: 2
- `published`: 0
- `not_started`: 84

Existing thin routes and reviewed seed profiles do not count as formally
published until bilingual page QA passes.

## Out of scope

- No new timetable acquisition.
- No raw local evidence.
- No page UI changes.
- No country profile expansion.
- No claim that any of the 98 pages is formally complete.

## Validation

```text
node scripts/check-country-page-programme.mjs
```

The dedicated workflow runs this validator whenever the contract, tracker,
country notes or validator changes.
