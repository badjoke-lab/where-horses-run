# PR-086 — JRA record-level official source verification

## Summary

PR-086 adds record-level official-source verification for the four existing JRA candidate records identified by PR-085. It verifies each current Tokyo/Kyoto candidate against official JRA 2026 calendar/program pages and records whether the stored first-race time matches the official JRA `発走時刻` evidence.

- `docs/runbooks/jra-record-level-source-verification.md`
- `scripts/check-jra-record-level-source-verification.mjs`
- `docs/runbooks/pr-086.md`
- `package.json`

## Records verified

- `japan-jra-2026-05-30-tokyo`
- `japan-jra-2026-05-30-kyoto`
- `japan-jra-2026-05-31-tokyo`
- `japan-jra-2026-05-31-kyoto`

## Official sources used

- `https://jra.jp/keiba/calendar2026/2026/5/0530.html`
- `https://jra.jp/keiba/calendar2026/2026/5/0531.html`

Only official JRA sources are used as authoritative evidence. The date pages are official JRA meeting-date/program pages and include the racecourse/date meeting labels plus first-race `発走時刻` values.

- `japan-jra-2026-05-30-kyoto`: official meeting date confirmed; first race time confirmed; comparison result `match`.
