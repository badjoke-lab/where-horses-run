# PR-060 — Japan inventory source expansion

## Summary

This PR expands the Japan racing inventory beyond JRA by adding official-source-based NAR racecourse inventory and Banei source candidates.

It does not add new timetable records.

## User-facing outcome

Japan can now be shown as a multi-system country with:

- JRA / central racing
- NAR / local government racing
- Banei / Obihiro

The NAR section now lists the official NAR racecourse guide entries for:

- Obihiro
- Monbetsu
- Morioka
- Mizusawa
- Urawa
- Funabashi
- Ohi
- Kawasaki
- Kanazawa
- Kasamatsu
- Nagoya
- Sonoda
- Himeji
- Kochi
- Saga

The Banei section now has a separate monthly schedule source candidate and remains mapped to Obihiro.

## Important limitation

This PR is inventory/source work only.

It does not make Japan covered.
It does not make NAR covered.
It does not make Banei covered.
It does not add NAR or Banei timetable records.
It does not add JRA coverage expansion.

Japan remains `initial_seed_only`.
NAR remains `not_started`.
Banei remains `not_started`.

## Safety boundary

This PR adds no live fetch, parser, source page fetching, raw HTML storage, response body storage, racecards, entries, horse names, jockey names, odds, results, payouts, prediction content, or tips.

## Validation
