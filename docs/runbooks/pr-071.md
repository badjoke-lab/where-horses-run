# PR-071 — Japan active-window refresh via generator

## Summary

This PR adds a deterministic reviewed Japan active-window candidate bundle generator.

It combines the existing JRA, NAR, and Banei candidate files into one reviewed candidate bundle.

## Files

- `scripts/generate-japan-active-window-approved-candidates.mjs`
- `scripts/check-japan-active-window-approved-candidates.mjs`
- `docs/runbooks/pr-071.md`

## User-facing outcome

No direct UI change in this PR.

This PR prepares the generator-based refresh path for Japan active-window data without directly replacing the public overlay yet.

## Safety boundary

This PR does not add live fetch.
This PR does not add source parsing.
This PR does not store source response bodies.
This PR does not store racecard, odds, results, payouts, predictions, or tips.
This PR does not directly publish a refreshed overlay.

The bundle is generated from existing safe candidate files:

- JRA candidates
- NAR candidates
- Banei candidates

## Validation

Run:

```sh
node scripts/generate-japan-active-window-approved-candidates.mjs
node scripts/check-japan-active-window-approved-candidates.mjs
npm run build
```

Package script wiring is intentionally not included in this PR because the direct package update was blocked during this pass.
