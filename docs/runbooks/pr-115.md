# PR-115 — USTA Harness full racecard Level A pattern

## Purpose

PR-115 continues the C/B-to-A phase after PR-114 established the first Level A pattern with Equibase Thoroughbred.

The goal is to apply the same Level A approach to the USTA Harness source family.

## Scope

Included:

- `data/fixtures/timetable/us-usta-racecard-full.html`
- `scripts/timetable/parsers/usta-full-racecard.mjs`
- `scripts/generate-pr-115-usta-level-a.mjs`
- `scripts/check-pr-115-usta-level-a.mjs`

## Level A rule

A record can claim Level A only when it contains:

- country
- group/system
- racecourse
- meeting date
- first race time
- full race rows
- each race number
- each race time
- source trace
- source capture date
- last checked date

The validator also requires `first_race_time` to match race 1 time.

## Source family

This PR uses USTA Harness because PR-113 already created B-level USTA records for Meadowlands and Yonkers.

## Non-goals

This PR does not add live fetch, complete United States full coverage, cover Equibase or AQHA Level A, add automated scheduled refresh, odds, payouts, predictions, tips, betting advice, or live coverage claims.
