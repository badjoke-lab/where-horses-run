# PR-113 — United States parser-backed promotion

## Purpose

PR-113 continues the 13-country D-to-C/B promotion work after PR-112.

The goal is to move the United States away from D-only source target status by adding fixture-backed promotion records for three source families:

- Equibase Thoroughbred
- USTA Harness
- AQHA Quarter Horse

## Scope

Included:

- `data/fixtures/timetable/us-equibase-entries.html`
- `data/fixtures/timetable/us-usta-entries.html`
- `data/fixtures/timetable/us-aqha-racing.html`
- `scripts/generate-pr-113-us-promotions.mjs`
- `scripts/check-pr-113-us-promotion.mjs`

## Data level handling

This PR does not claim Level A.

Rows with meeting date, racecourse, and first race time support Level B. Full race rows require source-specific racecard parsers before Level A can be claimed.

## User-facing meaning

The United States should no longer remain D-only in the 13-country major racing scope.

The generated rows can support display of:

- country
- system/group
- racecourse
- meeting date
- first race time
- source trace
- source capture date

## Known limits

The AQHA fixture uses neutral course labels in this PR because some real course-name fixture text was blocked during write operations. This keeps PR-113 focused on D-only removal and validator coverage. Later PRs can replace those neutral labels with reviewed real source rows while preserving the same schema.

## Non-goals

This PR does not add live fetch, complete United States full coverage, claim Level A, add automated scheduled refresh, odds, payouts, predictions, tips, betting advice, or live coverage claims.

## Next work

After PR-113, the next phase should move C/B records toward Level A through racecard and full race row parsers.
