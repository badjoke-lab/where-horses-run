# PR-056 — Japan real timetable seed

## Summary

PR-056 adds a conservative Japan real timetable seed to `data/generated/timetables.json`.

The seed is manual/source-reviewed only. It is intended to make Japan visible on the country, calendar, today, and tomorrow timetable pages when the record dates match the generated timetable range.

## Official JRA source used

Source id: `japan-jra-home`

Official source URLs reviewed for this seed:

- `https://jra.jp/keiba/calendar2026/2026/5/0530.html`
- `https://jra.jp/keiba/calendar2026/2026/5/0531.html`

The reviewed official JRA calendar pages identify the meeting dates, racecourses, and listed first race times used for the stored meeting-level records.

## Records added

Meeting-level Japan records were added for:

- Tokyo — 2026-05-30
- Kyoto — 2026-05-30
- Tokyo — 2026-05-31
- Kyoto — 2026-05-31

The records use:

- `country_id: japan`
- `source_id: japan-jra-home`
- `timezone: Asia/Tokyo`
- `racing_type: JRA meeting`
- `status: source-reviewed`
- `confidence: medium`

## Verification notes

The JRA calendar pages are treated as the official source of truth for this manual seed. The stored start time is limited to the listed first race time for each meeting.

Official JRA remains the final confirmation point. The source pages note that schedules and times can be changed by later official updates, weather, or other operating changes.

## Safety boundary

Live fetching remains disabled for this PR.

This PR does not add:

- a parser
- source page fetching
- source response body storage
- raw HTML storage
- racecard content
- entries
- horse names
- jockey names
- odds
- results
- payout data
- prediction content
- tips
- betting advice

Only safe meeting-level timetable fields are stored.

## Page behavior

Today, tomorrow, and 30-day calendar pages show these records only when the generated timetable date logic matches the record dates.

Country pages show Japan timetable records from the generated timetable data while continuing to direct users to official source links for final confirmation.
