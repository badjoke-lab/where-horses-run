# Static MVP release

Status: released static MVP baseline

Where Horses Run / 競馬どこ？ has reached the first static MVP release stage.

---

## Included in v0 static MVP

```text
- static Astro site
- English and Japanese root pages
- countries pages
- racecourses / tracks pages
- glossary pages
- sources page
- archive page
- fallback calendar pages
- about and disclaimer pages
- validation scripts
- accessibility and mobile pass
```

---

## Current product position

The project is currently:

```text
- static-first
- source-first
- low-runtime
- reference-oriented
- mobile-readable
- bilingual (English / Japanese)
```

The project is not yet:

```text
- live race tracking
- odds service
- betting platform
- race replay platform
- results database
- paid feed mirror
```
