# PR-055 — Hong Kong real timetable seed

## Summary

PR-055 adds the first Hong Kong real timetable seed to `data/generated/timetables.json`.

The seed is manual and source-reviewed. It uses the official HKJC fixture page for meeting-level dates, racecourse names, and day/twilight/night start-time-level indicators only.

## User-facing outcome

- The Hong Kong country page can show verified safe timetable record counts and record details.
- The 30-day calendar can show Hong Kong records when the seeded dates are inside the generated timetable range.
- The Today and Tomorrow pages continue to show records only when their generated dates exactly match seeded timetable record dates.
- Users are still directed to the official HKJC source for final confirmation.

## Source / verification notes

- Official source: HKJC local racing fixture page for June 2026.
- The records are meeting-level only.
- Exact first-race start times are not stored because this seed only verifies fixture-level day/twilight/night indicators.
- Happy Valley records currently store the safe racecourse name only; the racecourse ID requires later normalization because the static data only has a Sha Tin racecourse ID.

## Safety boundary

- Live fetching remains disabled.
- No parser is added.
- No source response body is stored.
- No racecard / entries / horse names / jockey names / odds / results / payout / prediction / tips are included.
- The file stores only safe timetable fields: country, racecourse name or existing ID, date, start-time-level text, timezone, racing type, source metadata, status, confidence, and notes.
