# PR-082 — Add official source evidence for existing countries

## Summary

PR-082 adds an official-source evidence runbook for the countries already introduced in the project: Japan, Hong Kong, and the United Arab Emirates. It documents what official sources were checked and what each source proves before any further country expansion.

## Files

- `docs/runbooks/existing-countries-official-source-evidence.md`
- `scripts/check-existing-countries-official-source-evidence.mjs`
- `docs/runbooks/pr-082.md`
- `package.json`

## User-facing outcome: documentation/evidence only

This PR is documentation and validation only. It does not change public timetable coverage, generated overlays, country pages, candidate timetable records, live fetch behavior, parsers, racecards, odds, results, payouts, predictions, tips, or runtime data.

## Why South Korea is paused

South Korea and all other new country work are paused because Japan, Hong Kong, and the UAE already exist in repo data but need stronger official source evidence. In particular, the UAE source review shows that the current three-racecourse inventory is not enough: official Emirates Racing sources identify five current UAE venues, so existing-country source evidence and inventory correction must happen before adding new countries.

## Official source evidence scope

The evidence runbook covers:

- Japan, split into JRA, NAR, and Banei.
- Hong Kong / HKJC.
- UAE / Emirates Racing.

For each country or racing system it records:

- racing authority/operator source;
- racecourse inventory source;
- fixture/calendar source;
- meeting date source;
- first race time source status, where available;
- timezone assumption source;
- known exclusions;
- gaps still requiring source confirmation;
- explicit repo inventory versus official source inventory comparison.
