# PR-057 — UAE real timetable seed

## Summary

PR-057 adds source-reviewed UAE timetable records to `data/generated/timetables.json`.
The records use only official Emirates Racing Authority source material and are limited to meeting-level timetable metadata.

## Official UAE source

- Source ID: `uae-era-home`
- Official source URL: `https://emiratesracing.com/season-calendar/current-season`
- The Emirates Racing Authority season calendar is the reviewed source for the UAE meeting dates in this seed.
- The official UAE source remains the final confirmation for any displayed meeting.

## Safety boundary

This PR keeps the timetable seed manual and source-reviewed only.

- Live fetching remains disabled.
- No parser is added.
- No source response body is stored.
- No raw HTML is stored.
- No racecard, entries, horse names, jockey names, odds, results, payout, prediction, or tips are included.
- Records are meeting-level only and do not include race-level content.

## User-facing outcome

The UAE records can appear on these pages when their dates match the generated timetable range:

- `/countries/united-arab-emirates/`
- `/calendar/`
- `/today/`
- `/tomorrow/`

Today, Tomorrow, and 30-day calendar views show records only when the record date falls in the relevant view window.

## Validation

Use Node v22.22.2 or a compatible version, then run:

```sh
npm run validate:timetable-schema-v0
npm run validate:uae-real-timetable-seed
npm run validate:japan-real-timetable-seed

```
