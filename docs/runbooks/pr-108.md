# PR-108 — Promote Australia / New Zealand timetable source targets

## Purpose

PR-108 is the first promotion PR after PR-107.

PR-107 established that D is only an extraction backlog for the 13-country major racing scope. PR-108 starts reducing that backlog by promoting Australia and New Zealand source targets to C or better where official meeting rows are available.

## Scope

Included:

- Australia / Racing Australia Thoroughbred: D to C
- New Zealand / HRNZ Harness: D to C and one A record where all race times are captured
- New Zealand / LOVERACING Thoroughbred: D to C proof records from official RaceInfo rows

Not fully completed in this PR:

- Harness Australia D to C remains pending because this PR did not safely capture a verified meeting row from the official fields page.
- LOVERACING C records in this PR are parser-proof records from official RaceInfo rows and should be supplemented with current/future programme rows in a later PR.

## User-facing meaning

This does not complete Australia/New Zealand timetable coverage.

It proves that D-only status was not an acceptable final state and that official meeting rows can promote major-country systems into usable C/A records.

## Operational meaning

The D-to-C promotion path is now concrete:

1. Start from PR-107 source target.
2. Capture official meeting rows.
3. Normalize them into the common best-available format.
4. Validate that records are A/B/C, not D.
5. Keep `promotes_from` pointing back to the PR-107 D target.
