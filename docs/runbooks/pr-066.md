# PR-066 — Safe source adapter interface

## Summary

This PR adds the shared TypeScript interface for safe timetable source adapters.

It is the interface layer needed before adding country/system generators such as:

- JRA
- NAR
- Banei
- HKJC
- ERA

## Files

- `src/lib/timetable/source-adapter.ts`
- `scripts/check-safe-source-adapter-interface.mjs`
- `docs/runbooks/pr-066.md`

## User-facing outcome

No direct UI change.

This PR prepares the generator architecture that will replace repeated manual seed editing.

## Safety boundary

The adapter interface locks the safety policy to:

- `storeSourceBody: false`
- `storeRawMarkup: false`
- `publishWithoutReview: false`
- `allowedOutput: meeting_level_only`

Adapters must generate candidate files only. They must not publish production-facing timetable overlay data directly.

## Validation

Run:

```sh
node scripts/check-safe-source-adapter-interface.mjs
npm run build
```

If package wiring is added later:

```sh
npm run validate:safe-source-adapter-interface
npm run check
```
