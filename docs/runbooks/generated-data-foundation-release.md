# Generated Data Foundation release

Status: released generated data foundation baseline
Phase: Generated Data Foundation

Where Horses Run / 競馬どこ？ has reached the generated data foundation release stage.

---

## Included in this phase

- generated data schema specification
- fetch status model specification
- source registry specification
- parser interface specification
- parser fixture test specification
- manual generated schedule seed
- generated data freshness display
- generated data fallback and empty states

---

## Current generated data position

The generated data layer is currently:

- static-first
- manual-seed based
- placeholder-labeled
- source-first
- parser-ready
- fixture-test-ready
- visible in calendar UI

The generated data layer is not yet:

- live fetched
- parser generated
- scheduled automatically
- a live racing feed
- a racecard redistribution layer
- an odds or result service

---

## Public data boundary

The project may show:

- generated status
- generated timestamps
- placeholder schedule entries
- source status
- source links
- freshness notes
- fallback calendar sections

The project must not republish:

- full racecards
- odds
- results
- payouts
- betting tips
- private or paid feeds
