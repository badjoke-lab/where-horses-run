# PR-286 — Define the country profile v2 contract

## Summary

This PR defines the structured bilingual content contract for all 98 country
and region detail pages.

It adds a versioned schema, validation modules, reference fixtures, a dedicated
workflow and the profile-v2 operating contract.

## Contract rules

- Country ID and slug must match.
- Reader-facing sections require separate English and Japanese content.
- Current country and special pages require racing types, racing systems and
  principal racecourse IDs.
- Organiser and distributor source IDs remain separate.
- Review dates cannot be future dates.
- Page kind must match the canonical 98-country tracker.
- Pending source research cannot claim a completed display ceiling.

## Programme impact

This infrastructure PR does not migrate production profiles and does not
advance any tracker row.
