# PR-295 — Profile v2 for entries 13–20

## Scope

This PR moves Japan, Hong Kong, New Zealand, South Africa, Uruguay, Sweden, Denmark, and the Czech Republic to reviewed profile-v2 records.

## Main changes

- migrate Japan and Hong Kong from the temporary legacy compatibility path
- remove legacy profile runtime support
- add profile-v2 records for all entries 13–20
- add country registry seeds for entries 15–20
- add official source registry records used by the new profiles
- load all new country, profile, and source batches through `src/lib/data.ts`
- advance entries 13–20 to `profile_ready`
- add profile, runtime, roadmap, source, note, programme, and publication validation

## Structural boundaries

- Japan remains separated into JRA, regional NAR racing, and Banei Tokachi.
- New Zealand remains separated into Thoroughbred and harness systems.
- South Africa remains separated into 4Racing and Gold Circle systems.
- Uruguay, Sweden, Denmark, and the Czech Republic remain source-led profiles without inferred principal racecourses.
