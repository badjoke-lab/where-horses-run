# Source-test scope 37–44

This wave records only official source roles, meeting-date and racecourse confirmation, technical schedule capability, coverage limits, and public display ceilings.

The wave does not copy complete programmes, racecards, participants, betting information, results, payouts, video, or raw official text.

Country-level completeness is assessed separately from the capability of an individual racecourse, authority, or racing-code subsystem.
