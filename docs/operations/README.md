# Operations documentation

Canonical operational rules for Where Horses Run:

- [`deployment-and-ci-policy.md`](./deployment-and-ci-policy.md) — Cloudflare Pages deployment control, commit prefixes, preview branch rules, GitHub Actions limits, and publication sequencing

## Required reading

Read the deployment and CI policy before:

- creating or renaming a development branch
- creating a pull request
- adding or changing a GitHub Actions workflow
- triggering a Cloudflare preview
- merging a non-publication PR
- merging a QA/publish PR

The policy applies across all project work, including the 98-country page programme.
